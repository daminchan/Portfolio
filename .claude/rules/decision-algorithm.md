# 判断アルゴリズム設計

新しい判断ルールを追加する際の構造化形式を定義する。

## 推奨形式: 階層的ルールベース + 排除チェックリスト

- **MUST**: IF-THEN ルール形式を使用する（LLM が最も正確に処理できる）
- **MUST**: MECE 原則（相互排他的・網羅的）に基づいて設計する
- **MUST**: 4レベル構造で記述する

## 4レベル構造

```
レベル1: 排除チェックリスト（1つでも該当すれば不使用）
レベル2: 肯定ルール（IF-THEN 形式）
レベル3: 核心的質問（判断の本質を問う質問1つ）
レベル4: 具体例（最小限のコード例）
```

## 新規ルール追加手順

1. 公式ドキュメント・ベストプラクティスを調査する
2. 排除条件 → 肯定条件 → 核心的質問 → 具体例の順で構造化する
3. MECE（条件の重複・漏れなし）を検証する
