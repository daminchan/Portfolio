# Implementation Workflow

## 絶対厳守

- **NEVER**: ユーザー確認なしに実装を開始しない
- **NEVER**: ユーザー承認なしにコミット・プッシュしない
- **MUST**: 段階的に進める（一気に実装しない）
- **MUST**: 実装前後に `@.claude/skills/implementation-check/SKILL.md` のチェックリストを実行する

## フロー

1. **理解確認** — 不明点があれば必ず質問
2. **実装宣言** — タスク・変更ファイル・方針を提示し、ユーザー確認
3. **実装前チェック** — implementation-check の実装前チェックリストを確認
4. **実装** — 都度進捗を表示
5. **実装後チェック** — implementation-check の実装後チェックリスト + 報告フォーマットで報告
6. **ビルド確認** — `npm run build` でエラーなし
7. **コミット確認** — 「コミット・プッシュしてよいですか？」と確認後に実行
